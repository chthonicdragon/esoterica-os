Задача: реализовать в приложении систему динамических оккультных тем, которые выбираются автоматически на основе архетипа пользователя, традиции и личных предпочтений.

Нужно создать полноценную Theme Engine систему.

1. Система архетипов пользователя

Создать систему, которая рассчитывает archetypeProfile пользователя на основе:

- выбранной традиции (например: греческая, демонолатрия, хаос-магия и т.д.)
- любимых божеств / духов
- используемых символов
- выбранных элементов (луна, ночь, свет, тьма, огонь)
- личных настроек темы

Пример структуры:

const archetypeProfile = {
  archetype: "witch_moon",
  tradition: "hellenic",
  preferences: ["moon","night","crossroads","forest"]
}

2. Theme Engine

Создать Theme Engine который выбирает тему автоматически:

function resolveTheme(profile) {
  if (profile.preferences.includes("crossroads")) return "hekate_crossroads"
  if (profile.preferences.includes("moon")) return "lunar_theme"
  if (profile.preferences.includes("light")) return "angelic_theme"
  if (profile.preferences.includes("dark")) return "gothic_theme"
  return "default_theme"
}

Тема должна применяться глобально ко всему приложению.

3. Создать несколько уникальных тем

Тема 1 — Lunar Theme

- мягкая ночная палитра
- луна светит сверху страницы
- свет луны создает легкий glow
- легкий туман
- плавная анимация облаков

Тема 2 — Hekate Crossroads

Главный dashboard заменяется на интерактивную сцену:

ночной лес  
перекресток дорог  
большая луна  

На перекрестке расположены кликабельные иконки разделов приложения:

- Алтарь
- Ритуалы
- Гримуар
- Паутинa знаний
- Оракул

Каждая дорога ведет к разделу приложения.

Добавить:

- мерцающие факелы
- легкий туман
- пара собак вдалеке
- звук ночного леса (опционально)

Тема 3 — Red Moon

- красная луна
- темное небо
- легкий красный glow
- более драматичная атмосфера
- редкие частицы пепла

Тема 4 — Angelic Light

- светлая тема
- мягкое золото
- лучи света
- прозрачные стеклянные панели
- спокойная минималистичная атмосфера

Тема 5 — Gothic Dark

- темная готическая тема
- черный + фиолетовый
- текстуры камня
- свечи
- медленное мерцание света

4. Анимации и эффекты

Использовать:

- CSS animations
- canvas particles
- subtle parallax
- glow effects

Эффекты должны быть легкими и не перегружать мобильные устройства.

5. Полная адаптация

Темы должны корректно работать:

- на мобильных устройствах
- на планшетах
- на desktop

Использовать responsive layout.

6. Архитектура тем

Создать структуру:

/themes
  lunar
  hekate
  redmoon
  angelic
  gothic

Каждая тема содержит:

theme.css  
theme.json  
background.ts  

Пример theme.json

{
  "name": "Lunar",
  "music": "lunar_theme.mp3",
  "colors": {
    "primary": "#6e7cff",
    "background": "#0b0f1f"
  }
}

7. Система музыкальных тем

Добавить возможность легко подключать музыку для каждой темы.

Функция:

loadThemeMusic(themeName)

Музыка должна:

- автоматически подгружаться
- иметь плавный fade
- отключаться в настройках

8. Панель переключения

Добавить настройки темы:

- автоматическая тема (по архетипу)
- выбор вручную
- включить / выключить музыку
- включить / выключить эффекты

9. Оптимизация

Важно:

анимации должны быть lightweight  
lazy loading для тяжелых эффектов  
fallback версия темы для слабых устройств

10. Код должен быть чистым и легко расширяемым.

Добавление новой темы должно требовать только:

- создать новую папку в /themes
- добавить theme.json
- добавить стили

Theme Engine должен автоматически подхватывать новую тему.
